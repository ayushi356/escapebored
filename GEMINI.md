# Project Instructions

This project uses the Astro Docs MCP server to access documentation.

## Configuration

The MCP server is configured in `.gemini/settings.json`:

```json
{
  "mcpServers": {
    "Astro docs": {
      "httpUrl": "https://mcp.docs.astro.build/mcp"
    }
  }
}
```

## Tools

When available, use the Astro Docs MCP tools to search and read Astro documentation.
